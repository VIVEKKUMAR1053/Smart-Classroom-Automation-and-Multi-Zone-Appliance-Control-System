# Smart-Classroom-Automation-and-Multi-Zone-Appliance-Control-System
IoT-based Smart Classroom Automation and Multi-Zone Appliance Control System designed to intelligently manage lights, fans, and electrical appliances across multiple classroom zones. The system enhances energy efficiency, reduces power consumption, enables automated device control, and improves classroom management through embedded systems 

## Overview
This project is an IoT-based Smart Classroom Automation System designed to intelligently control lights, fans, and electrical appliances across multiple classroom zones. The system improves energy efficiency, reduces power wastage, and enhances classroom management through automation.

## Objectives
- Automate classroom appliance control.
- Reduce unnecessary power consumption.
- Improve energy efficiency.
- Enable multi-zone appliance management.
- Provide a smart and scalable classroom solution.

## Key Features
- Multi-Zone Appliance Control
- Automatic Light and Fan Management
- PIR-Based Occupancy Detection
- Temperature-Based Fan Control
- Energy Efficient Operation
- Real-Time Monitoring and Automation

## Hardware Components
- Arduino Uno / ESP32
- PIR Motion Sensor
- Temperature Sensor
- Relay Module
- LED Lights
- Fan Control Circuit
- Power Supply Unit

## Technologies Used
